Overall System Architecture

Power Generation Service Components ESP32 & Sensors

Power Generation Overview

  • Bicycle dynamo generates 6W AC at 12V
  • Schottky rectifier converts AC to DC
  • Capacitor filters voltage ripple
  • TVS diode protects against surge
  • DC-DC converter regulates voltage
  • BMS safely charges Li-ion battery

Service Components Overview

MPU-6050

Detects sudden acceleration, tilt, and motion patterns using IMU data.

GPS Neo-6M

Provides real-time latitude and longitude for live tracking.

ESP32 Logic

Processes sensor data, detects events, packages JSON, and sends to server.
